## 🚀 Cài đặt nhanh (Quick Start)
Mở Terminal trên RPi hoặc Luckfox và dán dòng lệnh sau:

```bash
wget -q [https://raw.githubusercontent.com/Thooooooo/luckfox-store/main/fox-store.sh](https://raw.githubusercontent.com/Thooooooo/luckfox-store/main/fox-store.sh) -O fox-store.py && python3 fox-store.py
